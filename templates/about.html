Python Socket Web Server

Một web server đơn giản được xây dựng với Python Socket

Giới thiệu

Đây là một trang web mẫu được tạo cho web server Socket Python của bạn. Web server này sử dụng module socket trong Python để xử lý các request HTTP và phản hồi với nội dung HTML.

Socket server là một phương pháp mạnh mẽ để xây dựng các ứng dụng web từ cơ bản, giúp bạn hiểu sâu hơn về cách hoạt động của web server.

Tìm hiểu thêm

Tính năng

  • Xử lý các HTTP request đơn giản
  • Phục vụ nội dung HTML, CSS và JavaScript
  • Hỗ trợ các phương thức GET và POST
  • Khả năng mở rộng dễ dàng
  • Hiệu suất tốt cho các ứng dụng nhỏ

Mã nguồn đơn giản

import socket

HOST = '127.0.0.1'
PORT = 8080

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
server.bind((HOST, PORT))
server.listen(1)

print(f"Server running on http://{HOST}:{PORT}")

Hướng dẫn

Để chạy socket server:

  1. Lưu mã Python vào một file (ví dụ: server.py)
  2. Lưu nội dung HTML này vào một file (ví dụ: index.html)
  3. Chạy server với lệnh python server.py
  4. Truy cập server qua trình duyệt tại địa chỉ http://127.0.0.1:8080
